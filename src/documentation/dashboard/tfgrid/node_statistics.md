# Statistics

![](../img/statistics.png)

- Number of Nodes
- Number of Dedicated Nodes
- Number of Farms
- Number of Countries 
- The capacity CRU, SRU, HRU, MRU
- Number of GPUs
- Number of Gateways
- Number of Twins
- The number of public IPs available
- Number of Contracts
